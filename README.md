# AI Ad Campaign Dashboard 🚀

A modern, high-performance, and beautifully designed Ad Campaign Management module built for the **BizLeap Competitor Analyzer** platform. 

This application features a premium "Light Mode" aesthetic with Emerald and Teal accents, glassmorphism UI elements, and smooth micro-animations, providing a true "AI Operating System" feel.

## 🌟 Features
- **Animated Landing Page:** Includes beautiful gradient typography, floating blobs, and staggered animations.
- **Campaign Overview Dashboard:** Real-time metrics with interactive Recharts visualizations.
- **AI Recommendations Panel:** Simulates AI-generated insights for budget optimization and audience targeting.
- **Campaign Management Table:** Sortable, filterable campaign data with intuitive action menus.
- **Premium UI:** Built with Tailwind CSS, featuring soft drop-shadows, frosted glass (`backdrop-blur`) cards, and custom scrollbars.

## 💻 Tech Stack
- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Charts:** Recharts

## 🚀 How to Run Locally

Follow these steps to get the project running on your local machine:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/kurewarvaishnavi05/ai-ad-campaign-dashboard.git
   cd ai-ad-campaign-dashboard
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the Application**:
   Open your browser and navigate to: 👉 **[http://localhost:5173/](http://localhost:5173/)**
